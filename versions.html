Barry Cap › Version History

0.0

 The very first version of the site, when it was light and the header was ASCII, with a background image that was taking about 20 seconds to load… This version was initiated in 2020.

1.0

 The new look of the site, dark, with font Future Menu Brown.

 1.1

  New header was set on all pages.

 1.2

  Animated backgrounds were added, with different image for each page.

 1.3

  Spotlight image and page were created.

 1.4

  Images and videos in Home page that have been here since the beginning of the website, were put in Image and Video pages.
  SVG header was changed, using stroke instead of fill, and with a viewBox size that makes sense. Intro animations were also added, with a drawing one for the header.

 1.5

  Different sizes (for, texts, buttons, SVGs…) were set for different screen sizes, to make the website mobile friendly.

 1.6

  An intro page was added, and the old index.html was moved to new home.html.
