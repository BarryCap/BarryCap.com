Wavefront

 Wavefront is a descriptive language for coding 3D objects. The file format used for the shaping of objects using Wavefront is OBJ. It can be combined with an MTL file to give texture to surfaces, but we are going to use only basics of OBJ.

OBJ file explaination

Below is standing the content of the OBJ file (with comments).

v 1 0 1 #6: bottom right back
