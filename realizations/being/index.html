Description

A series by Barry Cap, created in the early 9th month of 2021.

This is about one being who is illustrated through moods and activities. They have a head, a neck and two arms. The pictures are monochrome, the being is white and the background is black.

The animtions are processed using CSS keyframes inside SVG. At the time this is written, the SVG files and this HTML file need to be opened in a browser (especially Chromium-based) to be well animated.
